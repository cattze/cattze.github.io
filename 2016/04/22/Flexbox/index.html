Flex布局-骰子

发表于 2016-04-22 | 分类于 CSS3

如果您需要在几分钟内构建复杂的CSS布局，该怎么办？ Flexbox是一种新的CSS布局规范，可以轻松构建动态布局。 使用flexbox，垂直居中，相同高度的列，重新排序和方向都是很容易的。

HTML模板。

<div class=“box“>
  <span class=“face“></span>
</div>

.box{
  margin: 16px;
  padding: 4px;
  background-color: #e7e7e7;
  width: 104px;
  height: 104px;
  box-shadow: inset 0 5px white, inset 0 -5px #bbb, inset 5px 0 #d7d7d7, inset -5px 0 #d7d7d7;
  border-radius: 10%;
}
.face{
  display: block;
  width: 24px;
  height: 24px;
  border-radius: 50%;
  margin: 4px;
  background-color: #333;
  box-shadow: inset 0 3px #111, inset 0 -3px #555;
}

div元素（代表骰子的一个面）是Flex容器，span元素（代表一个点）是Flex项目。如果有多个项目，就要添加多个span元素。

1个点的情况。

.box .first-face{
    display: flex;
    justify-content: center;
    align-items: center;
  }

2个点的情况。

.box .second-face{
    display: flex;
    justify-content: space-between;
  }
  .second-face .face:nth-child(2) {
    align-self: flex-end;
  }

3个点的情况。

.box .third-face{
    display: flex;
  }
  .third-face .face:nth-child(2) {
    align-self: center;
  }
  .third-face .face:nth-child(3) {
    align-self: flex-end;
  }

4个点的情况。

<div class=”box fourth-face”>
    <div class=”cloumn”>
    <span class=”face”><span>
    <span class=”face”><span>
    <div>
    <div class=”cloumn”>
    <span class=”face”><span>
    <span class=”face”><span>
    <div>
  <div>

.box .fourth-face{
    display: flex;
    justify-content: space-between;
    flex-direction: column;
  }
  .fourth-face .cloumn {
    display: flex;
    justify-content: space-between;
  }

5个点的情况。

<div class=”box fifth-face”>
    <div class=”cloumn”>
    <span class=”face”><span>
    <span class=”face”><span>
    <div>
    <div class=”cloumn”>
    <span class=”face”><span>
    <div>
    <div class=”cloumn”>
    <span class=”face”><span>
    <span class=”face”><span>
    <div>
  <div>

.box .fifth-face{
    display: flex;
  }
  .fifth-face .cloumn {
    display: flex;
    flex-direction: column;
    justify-content: space-between;
  }
  .fifth-face .cloumn:nth-child(2) {
    justify-content: center
  }

6个点的情况。

<div class=”box sixth-face”>
    <div class=”cloumn”>
    <span class=”face”><span>
    <span class=”face”><span>
    <span class=”face”><span>
    <div>
    <div class=”cloumn”>
    <span class=”face”><span>
    <span class=”face”><span>
    <span class=”face”><span>
    <div>
  <div>

.box .sixth-face{
    display: flex;
    justify-content: space-between;
  }
  .sixth-face .cloumn {
    display: flex;
    flex-direction: column;
    justify-content: space-between;
  }
