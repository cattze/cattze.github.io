SVG Symbols

发表于 2017-08-05 | 分类于 SVG

symbol 标记的作用是定义一个图像模板，你可以使用 use 标记实例化它，然后在SVG文档中反复使用，这种用法非常的高效。
symbol 本身不会输出任何图像，只有使用 use 实例化后才会显示。
use 标记的作用是能从SVG文档内部取出一个节点，克隆它，并把它输出到别处。跟‘引用’很相似，但它是深度克隆。


  <svg viewBox=”0 0 150 150” height=’300’>
    <svg style=”display: none;”>
      <symbol id=”sym01” viewBox=”0 0 150 110”>
        <circle cx=”50” cy=”50” r=”40” stroke-width=”8” stroke=”red” fill=”red”/>
        <circle cx=”90” cy=”60” r=”40” stroke-width=”8” stroke=”green” fill=”white”/>
      </symbol>
    </svg>
    <use xlink:href=”#sym01” x=”0” y=”0” width=”100” height=”50”/>
    <use xlink:href=”#sym01” x=”0” y=”50” width=”75” height=”38”/>
    <use xlink:href=”#sym01” x=”0” y=”100” width=”50” height=”25”/>
  </svg>

你定义了一组图形对象（使用 symbol 元素）之后，可以使用 use 元素来对它进行无限次实例化展示。
你使用xlink:href属性来指定你想要展示哪一组图标，这里，我们要展示的是id为#sym01 symbol 元素。
你可能注意到了我们给包裹 symbol 元素的SVG标签加了一个style=”display: none;”的样式：这是因为即使 symbol 本身没有显示，但是包裹它的 svg 元素依旧会渲染并占用一些页面空间，这就是为什么我们需要隐藏svg元素。
symbol.svg

  <svg>
    <symbol id=”sym01” viewBox=”0 0 150 110”>
      <circle cx=”50” cy=”50” r=”40” stroke-width=”8” stroke=”red” fill=”red”/>
      <circle cx=”90” cy=”60” r=”40” stroke-width=”8” stroke=”green” fill=”white”/>
    </symbol>
  </svg>

引入

  <svg>
    <use xlink:href=”img/symbol.svg#sym01”/>
  </svg>
