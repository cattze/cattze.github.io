CSS3 伪类选择器

发表于 2017-11-03 | 分类于 CSS3

一、锚点伪类
1､ :link 选择未被访问的链接，并设置其样式。:link 选择器不会设置已经访问过的链接的样式。

a:link{ background-color:yellow; }

2､ :visited 选择已访问的链接，并设置其样式。
a:visited{ background-color:yellow; }

3､ :active 选择活动链接，并设置其样式。当您在一个链接上点击时，它就会成为活动的（激活的）。
a:active{ background-color:yellow; }

4､ :hover 选择鼠标指针浮动在其上的元素，并设置其样式。:hover 选择器可用于所有元素，不只是链接。
a:hover{ background-color:yellow; }

5､ :focus 选择获得焦点的输入字段，并设置其样式。接收键盘事件或其他用户输入的元素都允许 :focus 选择器。
input:focus{ background-color:yellow; }

他们的先后顺序，要让他们遵守一个爱恨原则LoVe/HAte,也就是Link–visited–hover–active。这样样式才能生效。
二、元素状态
1､ :enabled 为所有 type=”text” 的已启用的 input 元素设置其样式。:enabled 选择器匹配每个已启用的元素（大多用在表单元素上）。
input[type=”text”]:enabled{ background-color: #ff0000; }

2､ :disabled 为所有 type=”text” 的被禁用的 input 元素设置其样式。:disabled 选择器匹配每个被禁用的元素（大多用在表单元素上）。
input[type=”text”]:disabled{ background-color: #dddddd; }

3､ :checked 为所有被选中的 input 元素设置其样式。:checked 选择器匹配每个已被选中的 input 元素（只用于单选按钮和复选框）。
input:checked{ background-color: #ff0000; }

三、nth选择器
1､ :first-child 选择属于其父元素的首个子元素的每个

元素，并为其设置样式。:first-child 选择器用于选取属于其父元素的首个子元素的指定选择器。

p:first-child{ background-color:yellow; }

2､ :last-child 指定属于其父元素的最后一个子元素的 p 元素设置样式。:last-child 选择器匹配属于其父元素的最后一个子元素的每个元素。p:last-child 等同于 p:nth-last-child(1)。
p:last-child{ background:#ff0000; }

3､ :first-of-type 指定父元素的首个 p 元素的样式。:first-of-type 选择器匹配属于其父元素的特定类型的首个子元素的每个元素，等同于 :nth-of-type(1)。
p:first-of-type{ background:#ff0000; }

4､ :last-of-type 指定父元素的最后一个 p 元素的样式。:last-of-type 选择器匹配属于其父元素的特定类型的最后一个子元素的每个元素，等同于 :nth-last-of-type(1)。
p:last-of-type{ background:#ff0000; }

5､ :only-of-type 指定属于父元素的特定类型的唯一子元素的每个 p 元素的样式。:only-of-type 选择器匹配属于其父元素的特定类型的唯一子元素的每个元素。
p:only-of-type{ background:#ff0000; }

6､ :only-child 规定属于其父元素的唯一子元素的每个 p 元素的样式。:only-child 选择器匹配属于其父元素的唯一子元素的每个元素。
p:only-child{ background:#ff0000; }

7､ :nth-child(n) 规定属于其父元素的第 n 个子元素的每个 p 元素的样式。:nth-child(n) 选择器匹配属于其父元素的第 N 个子元素，不论元素的类型。n 可以是数字、关键词或公式。
p:nth-child(2){ background:#ff0000; }

8､ :nth-last-child(n) 规定属于其父元素的第 n 个子元素的每个 p 元素，从最后一个子元素开始计数。
:nth-last-child(n) 选择器匹配属于其元素的第 N 个子元素的每个元素，不论元素的类型，从最后一个子元素开始计数。n 可以是数字、关键词或公式。
p:nth-last-child(2){ background:#ff0000; }
