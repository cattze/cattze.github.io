Canvas 线性

发表于 2017-05-13 | 分类于 canvas

1､lineCap 属性
lineCap 属性设置或返回线条末端线帽的样式。
注意：”round” 和 “square” 值会使线条略微变长。
butt 默认。向线条的每个末端添加平直的边缘。
round 向线条的每个末端添加圆形线帽。
square 向线条的每个末端添加正方形线帽。


  //butt
  ctx.lineWidth = 10;
  ctx.beginPath();
  ctx.lineCap = ‘butt’;
  ctx.moveTo(60, 30);
  ctx.lineTo(200, 30);
  ctx.fillText(‘butt’, 10, 35);
  ctx.stroke();
  //round
  ctx.lineWidth = 10;
  ctx.beginPath();
  ctx.lineCap = ‘round’;
  ctx.moveTo(60, 60);
  ctx.lineTo(200, 60);
  ctx.fillText(‘round’, 10, 65);
  ctx.stroke();
  //square
  ctx.lineWidth = 10;
  ctx.beginPath();
  ctx.lineCap = ‘square’;
  ctx.moveTo(60, 90);
  ctx.lineTo(200, 90);
  ctx.fillText(‘square’, 10, 95);
   ctx.stroke();


2､lineJoin 属性
lineJoin 属性设置或返回所创建边角的类型，当两条线交汇时。
注意：”miter” 值受 miterLimit 属性的影响。
bevel 创建斜角。
round 创建圆角。
miter 默认。创建尖角。


  //miter
  ctx.lineWidth = 10;
  ctx.beginPath();
  ctx.lineJoin = ‘miter’;
  ctx.moveTo(60, 30);
  ctx.lineTo(100, 30);
  ctx.lineTo(100, 100);
  ctx.fillText(‘miter’, 10, 35);
  ctx.stroke();
  //round
  ctx.lineWidth = 10;
  ctx.beginPath();
  ctx.lineJoin = ‘round’;
  ctx.moveTo(160, 30);
  ctx.lineTo(200, 30);
  ctx.lineTo(200, 100);
  ctx.fillText(‘round’, 120, 35);
  ctx.stroke();
  //bevel
  ctx.lineWidth = 10;
  ctx.beginPath();
  ctx.lineJoin = ‘bevel’;
  ctx.moveTo(260, 30);
  ctx.lineTo(300, 30);
  ctx.lineTo(300, 100);
  ctx.fillText(‘bevel’, 220, 35);
  ctx.stroke();


3､lineWidth 属性
lineWidth 属性设置或返回当前线条的宽度，以像素计。
线宽是指给定路径的中心到两边的粗细。就是在路径的两边各绘制线宽的一半。
在Canvas中绘制1个像素的线条时，坐标位置需要错开0.5个像素。
在Canvas中绘制路径（线段）时，后面显式设置的lineWidth会覆盖前面的值。
4､miterLimit 属性
默认值：10, number 正数。规定最大斜接长度。如果斜接长度超过 miterLimit 的值，边角会以 lineJoin 的 “bevel” 类型来显示。以最大斜接长度 5 绘制线条：


  ctx.lineWidth=10;
  ctx.lineJoin=”miter”;
  ctx.miterLimit=5;
  ctx.moveTo(20,20);
  ctx.lineTo(50,27);
  ctx.lineTo(20,34);
  ctx.stroke();


miterLimit 属性设置或返回最大斜接长度。
斜接长度指的是在两条线交汇处内角和外角之间的距离。

提示：只有当 lineJoin 属性为 “miter” 时，miterLimit 才有效。
边角的角度越小，斜接长度就会越大。
为了避免斜接长度过长，我们可以使用 miterLimit 属性。
如果斜接长度超过 miterLimit 的值，边角会以 lineJoin 的 “bevel” 类型来显示（Fig 3）
