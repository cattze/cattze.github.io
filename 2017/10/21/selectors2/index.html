CSS3 属性选择器selectors

发表于 2017-10-21 | 分类于 CSS3

属性选择器，是对带有指定属性的 HTML 元素设置样式。你可以指定元素的某个属性，还可以指定元素的某个属性和其对应的属性值。
一、[attribute] 用于选取带有指定属性的元素
选择带有 target 属性的 a 元素设置样式

a[target]{ background-color:yellow; }

你也可以使用多属性进行选择元素，title 和 target 同时具有这两属性的元素都将设置样式
a[title][target]{ background-color:yellow; }

二、[attribute=value] 用于选取带有指定属性和值的元素
选择带有 target=”_blank” 的 a 元素设置样式
a[target=_blank]{ background-color:yellow; }

这种属性选择器是属性值是一个或多个词列表，如果是列表时，他们需要用空格隔开，属性和属性值必须完全匹配。
三、[attribute~=value] 用于选取属性值中包含指定词汇的元素
选择 titile 属性包含单词 “flower” 的 a 元素，设置样式
a[title~=flower]{ background-color:yellow; }

这种属性选择器是属性值是一个或多个词列表，如果是列表时，他们需要用空格隔开，只要属性值中有一个value相匹配就可以选中该元素。
四、[attribute|=value] 用于选取带有以指定值开头的属性值的元素
选择 class 属性值以 “top” 开头的 a 元素设置样式
a[class|=top]{ background-color:yellow; }

该值必须是整个单词，比如 lang=”en”，或者后面跟着连字符，比如 lang=”en-us”。
这个选择器会选择 lang 属性值等于 en 或以 en- 开头的所有元素。
五、[attribute^=value] 匹配属性值以指定值开头的每个元素
选择 class 属性值以 “test” 开头的所有 a 元素设置样式，所选择的属性，其属性值是以 test 开头的所有元素。
a[class^=”test”]{ background:#ffff00; }

六、[attribute$=value] 匹配属性值以指定值结尾的每个元素
选择 class 属性值以 “test” 结尾的所有 a 元素设置样式，所选择的属性，其属性值是以 test 结尾的所有元素。
a[class$=”test”]{ background:#ffff00; }

七、[attribute=value] 匹配属性值包含指定值的每个元素
选择 class 属性值包含 “test” 的所有 a 元素设置样式，所选择的属性，其属性值中有这个 test 所有元素都将被选中。
a[class
=”test”]{ background:#ffff00; }
