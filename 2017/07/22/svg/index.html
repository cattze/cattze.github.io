SVG

发表于 2017-07-22 | 分类于 JavaScript

SVG 是英文 Scalable Vector Graphics 的缩写，意为可缩放矢量图。SVG 是用XML格式定义的矢量图。
相比起其它图片格式(比如JPEG和GIF)，SVG的优势有：
1､SVG 图片可以使用文本编辑器创建和编辑
2､SVG 图片能够实现内容搜索，索引，脚本控制和压缩
3､SVG 图片是可缩放的
4､SVG 图片可以以任意高分辨率打印
5､SVG 图片的缩放显示是无损的
6､SVG 是开放标准
7､VG 文件是纯XML的
最常见的元素：
一､rect:基础图形，基于角的位置和矩形的宽度和高度来创建的矩形。x:矩形距离左边的距离，y:矩形距离上边的距离，rx和ry矩形四个角的圆角效果。


  <svg width=”400” height=”110”>
    <rect x=”50” y=”20” rx=”20” ry=”20” width=”300” height=”100” style=”fill:rgb(0,0,255);stroke-width:3;stroke:rgb(0,0,0)” />
  </svg>

二、circle:基础图形，基于中心点以及半径来创建的圆形。cx和cy圆心的坐标，r:圆的半径长度。


  <svg height=”100” width=”100”>
    <circle cx=”50” cy=”50” r=”40” stroke=”black” stroke-width=”3” fill=”red” />
  </svg>

三、ellipse:绘制一个椭圆，椭圆跟圆形很相似。不同之处在于椭圆有两个半径，并且这两个值不同。cx:椭圆的x坐标，cy:椭圆的y坐标，rx:椭圆的横向半径，ry:椭圆的纵向半径。


  <svg height=”140” width=”500”>
    <ellipse cx=”240” cy=”100” rx=”220” ry=”30” style=”fill:purple” />
    <ellipse cx=”220” cy=”70” rx=”190” ry=”20” style=”fill:lime” />
  </svg>

四、line:绘制一条直线。x1:直线的x轴起始坐标，y1:直线的y轴起始坐标，x2:直线的x轴终止坐标，y2:直线的y轴终止坐标。


  <svg height=”210” width=”500”>
    <line x1=”0” y1=”0” x2=”200” y2=”200” style=”stroke:rgb(255,0,0);stroke-width:2”/>
  </svg>

五、polygon:基础图形，基于坐标点和两点相连而成的直线线段来创建的多边形。比如三角形，四边形，五边形等。points属性里定义了各个点的坐标，x和y坐标之间用逗号分别，多个坐标之间用空格分割。


  <svg height=”180” width=”500”>
    <polyline points=”0,40 40,40 40,80 80,80 80,120 120,120 120,160” style=”fill:none;stroke:black;stroke-width:3” />
  </svg>


  <svg height=”210” width=”500”>
    <polygon points=”100,10 40,198 190,78 10,78 160,198” style=”fill:lime;stroke:purple;stroke-width:5;fill-rule:evenodd;” />
  </svg>

六、path:万能的图形，可以用来创建任意的形状。所有基础图形都能由它来创建。
M x y：移动虚拟画笔到指定的（x,y）坐标，仅移动不绘制。
L x y：从当前画笔所在位置绘制一条直线到指定的（x,y）坐标。
H x：绘制一条水平直线到参数指定的x坐标点，y坐标为画笔的y坐标。
V y：从当前位置绘制一条垂直直线到参数指定的y坐标。
Z：从结束点绘制一条直线到开始点，闭合路径。


  <svg height=”210” width=”500”>
    <path d=”M150 0 L75 200 L225 200 Z” />
  </svg>

定义了一个路径，从点 150,0 开始，连直线到点 75,200，然后连直线到点 225,200，最后闭合这个路径，连直线回到点 150,0。
