Echarts的用法

发表于 2017-07-10 | 分类于 JavaScript

ECharts，一个纯 Javascript 的图表库，依赖轻量级的 Canvas 类库 ZRender，提供直观，生动，可交互，可高度个性化定制的数据可视化图表。
通过 npm 获取 echarts，npm install echarts –save。

常用的组件：
1､ color: 调色盘颜色列表。如果系列没有设置颜色，则会依次循环从该列表中取颜色作为系列颜色。

color: [‘#c23531’,’#2f4554’, ‘#61a0a8’, ‘#d48265’, ‘#91c7ae’,’#749f83’,  ‘#ca8622’, ‘#bda29a’,’#6e7074’, ‘#546570’, ‘#c4ccd3’]

2､ title: 标题组件，包含主标题和副标题。
title: {
  text: 标题
  show: 是否显示组件
  textStyle: { … } 标题文字的样式
  subtext: 副标题文本，支持使用 \n 换行
  subtextStyle: { … } 副标题文字的样式
}

3､ legend: 展现了不同系列的标记(symbol)，颜色和名字。
legend: {
  type: ‘plain’普通图例 ／ ‘scroll’可滚动翻页的图例
  show: 是否显示组件
  orient: ‘horizontal’水平 / ‘vertical’垂直
  textStyle: { … } 公用文本样式
  data: [{ name: ‘系列1’, // 强制设置图形为圆。 icon: ‘circle’, // 设置文本为红色 textStyle: { color: ‘red’ } }]
  //数据数组。数组项通常为一个字符串，每一项代表一个系列的 name（如果是饼图，也可以是饼图单个数据的 name）。图例组件会自动根据对应系列的图形标记（symbol）来绘制自己的颜色和标记，特殊字符串 ‘’（空字符串）或者 ‘\n’（换行字符串）用于图例的换行
}

4､ grid: 直角坐标系内绘图网格。单个 grid 内最多可以放置上下两个 X 轴，左右两个 Y 轴。可以在网格上绘制折线图，柱状图，散点图（气泡图）。
grid: {
  show: 是否显示组件
  borderWidth: 网格的边框线宽
  tooltip: { … } 提示框组件
}

5､ xAxis: 直角坐标系 grid 中的 x 轴，一般情况下单个 grid 组件最多只能放上下两个 x 轴，多于两个 x 轴需要通过配置 offset 属性防止同个位置多个 x 轴的重叠。
xAxis: {
  show: 是否显示组件
  type: ‘value’数值轴，适用于连续数据 / ‘category’类目轴，适用于离散的类目数据，为该类型时必须通过 data 设置类目数据 / ‘time’时间轴，适用于连续的时序数据，与数值轴相比时间轴带有时间的格式化，在刻度计算上也有所不同，例如会根据跨度的范围来决定使用月，星期，日还是小时范围的刻度 / ‘log’对数轴。适用于对数数据
  splitNumber: 坐标轴的分割段数，需要注意的是这个分割段数只是个预估值，最后实际显示的段数会在这个基础上根据分割后坐标轴刻度显示的易读程度作调整
  interval: 强制设置坐标轴分割间隔。因为 splitNumber 是预估的值，实际根据策略计算出来的刻度可能无法达到想要的效果，这时候可以使用 interval 配合 min、max 强制设定刻度划分，一般不建议使用。无法在类目轴中使用。在时间轴（type: ‘time’）中需要传时间戳，在对数轴（type: ‘log’）中需要传指数值。
  nameTextStyle: { … } 坐标轴名称的文字样式
  axisLine: { … lineStyle: { … } 坐标轴线线的样式 } 坐标轴轴线相关设置 
  axisTick: { … lineStyle: { … } 刻度线的样式 } 坐标轴刻度相关设置
  splitLine: {  //坐标轴在 grid 区域中的分隔线 
    interval: 坐标轴分隔线的显示间隔，设置成 0 强制显示所有标签，设置为 1，表示『隔一个标签显示一个标签』，如果值为 2，表示隔两个标签显示一个标签，以此类推 
    lineStyle: { … } 分隔线的样式
  }
  axisLabel: { //坐标轴刻度标签的相关设置 
    interval: 坐标轴刻度标签的显示间隔，默认会采用标签不重叠的策略间隔显示标签。设置成 0 强制显示所有标签。如果设置为 1，表示『隔一个标签显示一个标签』，如果值为 2，表示隔两个标签显示一个标签，以此类推
    formatter: 刻度标签的内容格式器，支持字符串模板和回调函数两种形式。 
    // 使用字符串模板，模板变量为刻度默认标签 {value} formatter: ‘{value} kg’
    // 使用函数模板，函数参数分别为刻度数值（类目），刻度的索引
        formatter: function (value, index) {
        // 格式化成月/日，只在第一个刻度显示年份
            var date = new Date(value);
            var texts = [(date.getMonth() + 1), date.getDate()];
            if (index === 0) {  ；‘
                texts.unshift(date.getYear());
            }
            return texts.join(‘/‘);
        }
    
  }
  data: //类目数据，在类目轴（type: ‘category’）中有效 1､所有类目名称列表 data: [‘周一’, ‘周二’, ‘周三’, ‘周四’, ‘周五’, ‘周六’, ‘周日’] 2､  每一项也可以是具体的配置项，此时取配置项中的 value 为类目名
    data: [{
        value: ‘周一’,
        // 突出周一
        textStyle: {
            fontSize: 20,
            color: ‘red’
        }
    }, ‘周二’, ‘周三’, ‘周四’, ‘周五’, ‘周六’, ‘周日’]
  
}

6､ yAxis: 直角坐标系 grid 中的 y 轴，一般情况下单个 grid 组件最多只能放左右两个 y 轴，多于两个 y 轴需要通过配置 offset 属性防止同个位置多个 Y 轴的重叠。
yAxis: {
  同xAxis
}

7､ series: 系列列表。每个系列通过 type 决定自己的图表类型
series: [{
  type: line | bar | pie 
}]
